Detailed Description

Utilities for working with SEGY files.

Author
Cathal O Broin - catha.nosp@m.l@ic.nosp@m.hec.i.nosp@m.e - first commit
Copyright
TBD. Do not distribute
Date

Utilities for working with SEGY files. Namely, labels for the number formats, offset for certain file header values, functions for working with scaled numbers, and functions for manipulating the parameter structure from buffers of trace headers.
