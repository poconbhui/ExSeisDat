Detailed Description

Functions for converting between ASCII and EBCDIC encoding.

We define a list of ASCII / EBCDIC pairs of characters. We use this list to create a map from ASCII characters to the equivalent EBCDIC character, and the reverse map, EBCDIC to ASCII. We then provide functions for these map lookups.
