Write the traces specified by the offsets in the passed offset array.

Parameters
[in]szThe number of traces to process
[in]offsetAn array of trace numbers to write.
[in]traceA contiguous array of each trace (size sz*ns*sizeof(exseis::utils::Trace_value))
[in]prmA parameter structure
[in]skipWhen writing, skip the first "skip" entries of prm

When prm==PIOL_PARAM_NULL only the trace DF is written. It is assumed that the parameter writing operation is not an update. Any previous contents of the trace header will be overwritten.
