Detailed Description

template<typename T>
class exseis::utils::Distributed_vector< T >::Concept

An abstract interface for implementing the Distributed_vector.

This class is used to implement the Type Erasure used in Distributed_vector.

Classes deriving from Distributed_vector should derive from this class and put their actual implementation in here. A specialized instance of this Concept class should then be passed to the Distributed_vector constructor during construction of a child class. The Distributed_vector class will then call the virtual member functions of this class.

This will be held in a std::unique_ptr in the Distributed_vector instance. This allows for specialization of the Distributed_vector class, along with value semantics while passing it around.
